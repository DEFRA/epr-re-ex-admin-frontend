# Project organisation

## Client

This is mostly defined by the template and there is probably little need to get creative with it.

```
project-root/
└── src/
    └── client/
        ├── common/
        │   └── ...content structure defined by the template
        ├── javascripts/
        │   └── ...content structure defined by the template
        └── stylesheets/
            └── ...content structure defined by the template
```
